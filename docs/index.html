1.2.6.1 함수 정의 문법

function(인자1, 인자2, 인자3, ..., 인자n) {
  표현식1
  표현식2
  ... 표현식n
  return(반환값)
}

1.2.6.2 함수 정의 예시

  • 하나의 숫자를 입력받아 제곱값을 계산하는 함수를 생성해 보자.
fun1<-function(num) {
  return(num ^ 2)
}
fun1(2)
## [1] 4
  • 여러 숫자를 입력받아 합을 출력하는 함수를 생성해 보자.
fun2<-function(...) {
  x<-sum(...)
  paste("합계: ", x)
}
fun2(2, 4, 6, 8, 10)
## [1] "합계:  30"
